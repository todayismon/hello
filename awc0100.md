## A. 兼职工资计算

### 中文题面
高桥君这个月做了 \(N\) 种兼职。第 \(i\) 种兼职的时薪是 \(A_i\) 日元，做了 \(T_i\) 小时。  
求他这个月总共能拿到多少工资。

### 思路
直接累加 \(\sum A_i \times T_i\) 即可。

### Python 代码
```python
import sys

input = sys.stdin.readline
n = int(input())
ans = 0
for _ in range(n):
    a, t = map(int, input().split())
    ans += a * t
print(ans)
```

## C. 权限管理系统

### 中文题面
有 \(N\) 个资源，每个资源对应一个长度为 \(L\) 的二进制串。  
如果员工权限串按位包含某资源所需权限，则该员工可访问该资源。  
每次询问给出若干资源编号，求能访问这些资源的最小权限串。

### 思路
要访问一组资源，所需权限就是这些资源权限串的按位 OR。

### Python 代码
```python
import sys

input = sys.stdin.readline
N, L, Q = map(int, input().split())
s = [int(input().strip(), 2) for _ in range(N)]

for _ in range(Q):
    tmp = list(map(int, input().split()))
    res = 0
    for x in tmp[1:]:
        res |= s[x - 1]
    print(bin(res)[2:].zfill(L))
```

## E. Mountain View

### 中文题面
给你 \(N\) 座山的高度 \(A_i\)。  
对每座山 \(i\)，输出有多少座山严格高于它。

### 思路
按高度排序，从高到低扫，处理同高度分组。

### Python 代码
```python
import sys

input = sys.stdin.readline
n = int(input())
a = list(map(int, input().split()))

pairs = sorted([(x, i) for i, x in enumerate(a)], reverse=True)
ans = [0] * n

higher = 0
i = 0
while i < n:
    j = i
    while j < n and pairs[j][0] == pairs[i][0]:
        j += 1
    for k in range(i, j):
        _, idx = pairs[k]
        ans[idx] = higher
    higher += (j - i)
    i = j

print(*ans)
```

## G. Circle of Friends

### 中文题面
给定无向图，询问每个点所在连通块的大小。

### 思路
并查集维护连通块大小。

### Python 代码
```python
import sys

input = sys.stdin.readline

class DSU:
    def __init__(self, n):
        self.p = list(range(n))
        self.sz = [1] * n

    def find(self, x):
        while self.p[x] != x:
            self.p[x] = self.p[self.p[x]]
            x = self.p[x]
        return x

    def union(self, a, b):
        a = self.find(a)
        b = self.find(b)
        if a == b:
            return
        if self.sz[a] < self.sz[b]:
            a, b = b, a
        self.p[b] = a
        self.sz[a] += self.sz[b]

n, m = map(int, input().split())
dsu = DSU(n)

for _ in range(m):
    u, v = map(int, input().split())
    dsu.union(u - 1, v - 1)

q = int(input())
for _ in range(q):
    s = int(input()) - 1
    print(dsu.sz[dsu.find(s)])
```

## H. 图的访问序

### 中文题面
给定有向图和每个点的重要度，按“可达且未访问点中重要度最大”的规则遍历，输出访问顺序。

### 思路
大根堆维护当前可选点；空了就找下一个未访问点作为新起点。

### Python 代码
```python
import sys
import heapq

input = sys.stdin.readline
n, m = map(int, input().split())
b = list(map(int, input().split()))

g = [[] for _ in range(n)]
for _ in range(m):
    u, v = map(int, input().split())
    g[u - 1].append(v - 1)

used = [False] * n
pq = []
ans = []

heapq.heappush(pq, (-b[0], 0))

for _ in range(n):
    while pq and used[pq[0][1]]:
        heapq.heappop(pq)
    if not pq:
        mx = -1
        idx = -1
        for i in range(n):
            if not used[i] and b[i] > mx:
                mx = b[i]
                idx = i
        heapq.heappush(pq, (-b[idx], idx))
    _, x = heapq.heappop(pq)
    if used[x]:
        continue
    used[x] = True
    ans.append(x + 1)
    for y in g[x]:
        if not used[y]:
            heapq.heappush(pq, (-b[y], y))

print(*ans)
```

## I. Circle Pass Game

### 中文题面
有 \(N\) 个孩子围成一圈，初始持球者是 \(S\)。

进行 \(M\) 次操作。第 \(i\) 次操作时：

1. 从当前持球者的下一个人开始，顺时针数还在圈内的人
2. 球传给第 \(D_i\) 个被数到的人
3. 当前持球者离开圈

输出最后持球者编号。

### 思路
用树状数组维护“谁还在圈内”，支持：

- 删除一个人
- 找第 \(k\) 个还在的人

### Python 代码
```python
import sys

input = sys.stdin.readline

class BIT:
    def __init__(self, n):
        self.n = n
        self.bit = [0] * (n + 1)

    def add(self, i, v):
        while i <= self.n:
            self.bit[i] += v
            i += i & -i

    def sum(self, i):
        s = 0
        while i > 0:
            s += self.bit[i]
            i -= i & -i
        return s

    def kth(self, k):
        idx = 0
        bitmask = 1 << (self.n.bit_length() - 1)
        while bitmask:
            t = idx + bitmask
            if t <= self.n and self.bit[t] < k:
                idx = t
                k -= self.bit[t]
            bitmask >>= 1
        return idx + 1

N, M, S = map(int, input().split())
D = [int(input()) for _ in range(M)]

bit = BIT(N)
for i in range(1, N + 1):
    bit.add(i, 1)

cur = S
for d in D:
    bit.add(cur, -1)
    rem = N - 1
    after = bit.sum(cur)
    k = d % rem
    if k == 0:
        k = rem
    if after >= k:
        cur = bit.kth(bit.sum(cur) + k)
    else:
        cur = bit.kth(k - after)

print(cur)
```

## J. Road Network Development

### 中文题面
给定一棵树。每次操作把一条路径上的边权全部加 1。  
之后回答若干次路径最小边权查询。

### 思路
重链剖分 + 线段树区间加、区间最小值。

### Python 代码
```python
import sys
sys.setrecursionlimit(1 << 25)
input = sys.stdin.readline

N = int(input())
P = [0] * (N + 1)
W = [0] * (N + 1)
g = [[] for _ in range(N + 1)]
for i in range(2, N + 1):
    p, w = map(int, input().split())
    P[i] = p
    W[i] = w
    g[p].append(i)

Q = int(input())
updates = [tuple(map(int, input().split())) for _ in range(Q)]
R = int(input())
queries = [tuple(map(int, input().split())) for _ in range(R)]

parent = [0] * (N + 1)
depth = [0] * (N + 1)
size = [0] * (N + 1)
heavy = [0] * (N + 1)

def dfs1(v):
    size[v] = 1
    mx = 0
    for to in g[v]:
        parent[to] = v
        depth[to] = depth[v] + 1
        dfs1(to)
        size[v] += size[to]
        if size[to] > mx:
            mx = size[to]
            heavy[v] = to

top = [0] * (N + 1)
pos = [0] * (N + 1)
cur = 0

def dfs2(v, tp):
    global cur
    top[v] = tp
    cur += 1
    pos[v] = cur
    if heavy[v]:
        dfs2(heavy[v], tp)
        for to in g[v]:
            if to != heavy[v]:
                dfs2(to, to)

dfs1(1)
dfs2(1, 1)

class SegTree:
    def __init__(self, n):
        self.N = 1
        while self.N < n:
            self.N <<= 1
        self.inf = 10**18
        self.mn = [self.inf] * (2 * self.N)
        self.lz = [0] * (2 * self.N)

    def build(self, arr):
        for i, x in enumerate(arr, 1):
            self.mn[self.N + i - 1] = x
        for i in range(self.N - 1, 0, -1):
            self.mn[i] = min(self.mn[i << 1], self.mn[i << 1 | 1])

    def push(self, i):
        if self.lz[i]:
            for c in (i << 1, i << 1 | 1):
                self.mn[c] += self.lz[i]
                self.lz[c] += self.lz[i]
            self.lz[i] = 0

    def add(self, l, r, v, i=1, tl=1, tr=None):
        if tr is None:
            tr = self.N
        if l > tr or r < tl:
            return
        if l <= tl and tr <= r:
            self.mn[i] += v
            self.lz[i] += v
            return
        self.push(i)
        tm = (tl + tr) >> 1
        self.add(l, r, v, i << 1, tl, tm)
        self.add(l, r, v, i << 1 | 1, tm + 1, tr)
        self.mn[i] = min(self.mn[i << 1], self.mn[i << 1 | 1])

    def query(self, l, r, i=1, tl=1, tr=None):
        if tr is None:
            tr = self.N
        if l > tr or r < tl:
            return self.inf
        if l <= tl and tr <= r:
            return self.mn[i]
        self.push(i)
        tm = (tl + tr) >> 1
        return min(self.query(l, r, i << 1, tl, tm),
                   self.query(l, r, i << 1 | 1, tm + 1, tr))

seg = SegTree(N)
base = [0] * (N + 1)
for i in range(2, N + 1):
    base[pos[i]] = W[i]
seg.build(base[1:])

def path_add(u, v, val):
    while top[u] != top[v]:
        if depth[top[u]] < depth[top[v]]:
            u, v = v, u
        seg.add(pos[top[u]], pos[u], val)
        u = parent[top[u]]
    if depth[u] > depth[v]:
        u, v = v, u
    if u != v:
        seg.add(pos[u] + 1, pos[v], val)

def path_min(u, v):
    if u == v:
        return 0
    res = 10**18
    while top[u] != top[v]:
        if depth[top[u]] < depth[top[v]]:
            u, v = v, u
        res = min(res, seg.query(pos[top[u]], pos[u]))
        u = parent[top[u]]
    if depth[u] > depth[v]:
        u, v = v, u
    if u != v:
        res = min(res, seg.query(pos[u] + 1, pos[v]))
    return res

for u, v in updates:
    path_add(u, v, 1)

out = []
for a, b in queries:
    out.append(str(path_min(a, b)))
print("\n".join(out))
```

## K. 商店街的区块选择

### 中文题面
给定长度为 \(N\) 的数组 \(A\) 和二进制数组 \(C\)。

- \(A_i\)：第 \(i\) 个区块的租金
- \(C_i\in\{0,1\}\)：第 \(i\) 个区块是否有停车位

你要维护长度恰好为 \(K\) 的连续区间。

有 \(Q\) 次操作：

- `1 X Y`：把 \(A_X\) 改成 \(Y\)
- `2 X`：在所有长度恰好为 \(K\) 的连续区间中，找出“停车位数量至少为 \(X\)”的区间里租金和最小值；若不存在，输出 `IMPOSSIBLE`

### 思路
对每个长度为 \(K\) 的窗口，维护：

- 窗口内租金和
- 窗口内停车位数

单点修改 \(A_X\) 时，只会影响最多 \(K\) 个窗口。  
把所有窗口分块维护：

- 每块内按停车位数排序
- 查询时用二分找到满足条件的后缀最小值

### 完整 Python 代码
```python
import sys
from bisect import bisect_left

input = sys.stdin.readline
INF = 10**30

N, K, Q = map(int, input().split())
A = list(map(int, input().split()))
C = list(map(int, input().split()))

W = N - K + 1
win_sum = [0] * W
win_cnt = [0] * W

s = sum(A[:K])
c = sum(C[:K])
win_sum[0] = s
win_cnt[0] = c
for i in range(1, W):
    s += A[i + K - 1] - A[i - 1]
    c += C[i + K - 1] - C[i - 1]
    win_sum[i] = s
    win_cnt[i] = c

B = 700
nb = (W + B - 1) // B

lazy = [0] * nb
block_cnts = [[] for _ in range(nb)]
block_mins = [[] for _ in range(nb)]

def rebuild(bid):
    l = bid * B
    r = min(W, (bid + 1) * B)
    arr = [(win_cnt[i], win_sum[i]) for i in range(l, r)]
    arr.sort()
    block_cnts[bid] = [x for x, _ in arr]
    suf = [INF] * (len(arr) + 1)
    for i in range(len(arr) - 1, -1, -1):
        suf[i] = min(suf[i + 1], arr[i][1])
    block_mins[bid] = suf

for bid in range(nb):
    rebuild(bid)

def range_add(l, r, delta):
    bl = l // B
    br = r // B
    if bl == br:
        for i in range(l, r + 1):
            win_sum[i] += delta
        rebuild(bl)
        return

    end = (bl + 1) * B - 1
    for i in range(l, end + 1):
        win_sum[i] += delta
    rebuild(bl)

    for bid in range(bl + 1, br):
        lazy[bid] += delta

    start = br * B
    for i in range(start, r + 1):
        win_sum[i] += delta
    rebuild(br)

def query(x):
    ans = INF
    for bid in range(nb):
        cnts = block_cnts[bid]
        if not cnts:
            continue
        idx = bisect_left(cnts, x)
        if idx == len(cnts):
            continue
        ans = min(ans, lazy[bid] + block_mins[bid][idx])
    return ans

for _ in range(Q):
    t, x, y = map(int, input().split())
    if t == 1:
        x -= 1
        delta = y - A[x]
        A[x] = y
        l = max(0, x - K + 1)
        r = min(x, W - 1)
        range_add(l, r, delta)
    else:
        ans = query(x)
        print(ans if ans < INF else "IMPOSSIBLE")
```

## L. Schedule Adjustment

### 中文题面
每个人有两个可能开始时间 \(A_i\) 和 \(A_i+D_i\)。  
选择一种安排，使给定冲突对之间的最小时间差尽量大；若有多种，输出字典序最小方案。

### 思路
二分答案 + 2-SAT 判断 + 构造字典序最小解。

### Python 代码
```python
import sys
sys.setrecursionlimit(1 << 25)
input = sys.stdin.readline

N, M = map(int, input().split())
A = [0] * N
D = [0] * N
for i in range(N):
    A[i], D[i] = map(int, input().split())

pairs = [tuple(map(int, input().split())) for _ in range(M)]
pairs = [(u - 1, v - 1) for u, v in pairs]

def var(i, val):
    return 2 * i + val

def sat(K, fixed=None):
    n2 = 2 * N
    g = [[] for _ in range(n2)]
    rg = [[] for _ in range(n2)]

    def add_imp(a, b):
        g[a].append(b)
        rg[b].append(a)

    def add_clause(a, av, b, bv):
        na = var(a, 1 - av)
        nb = var(b, 1 - bv)
        add_imp(na, var(b, bv))
        add_imp(nb, var(a, av))

    for u, v in pairs:
        for xu in (0, 1):
            tu = A[u] + xu * D[u]
            for xv in (0, 1):
                tv = A[v] + xv * D[v]
                if abs(tu - tv) < K:
                    add_clause(u, 1 - xu, v, 1 - xv)

    if fixed:
        for i, val in fixed.items():
            add_clause(i, val, i, val)

    order = []
    vis = [False] * n2

    def dfs1(v):
        vis[v] = True
        for to in g[v]:
            if not vis[to]:
                dfs1(to)
        order.append(v)

    comp = [-1] * n2

    def dfs2(v, c):
        comp[v] = c
        for to in rg[v]:
            if comp[to] == -1:
                dfs2(to, c)

    for i in range(n2):
        if not vis[i]:
            dfs1(i)

    c = 0
    for v in reversed(order):
        if comp[v] == -1:
            dfs2(v, c)
            c += 1

    for i in range(N):
        if comp[var(i, 0)] == comp[var(i, 1)]:
            return False
    return True

lo, hi = 0, 2 * 10**9 + 1
while lo + 1 < hi:
    mid = (lo + hi) // 2
    if sat(mid):
        lo = mid
    else:
        hi = mid

K = lo

fixed = {}
S = []
for i in range(N):
    fixed[i] = 0
    if sat(K, fixed):
        S.append('0')
    else:
        fixed[i] = 1
        S.append('1')

print(K)
print(''.join(S))
```

## M. Secret Sequence and Branching Notes

### 中文题面
维护很多版本的区间和模 \(K\) 断言。  
每次操作会在某个旧版本上新增断言或查询某区间和是否唯一确定。

### 思路
前缀和 + 带权并查集 + 版本树回滚。

### Python 代码
```python
import sys
sys.setrecursionlimit(1 << 25)
input = sys.stdin.readline

N, K, Q = map(int, input().split())

queries = [None] * (Q + 1)
children = [[] for _ in range(Q + 1)]
for i in range(1, Q + 1):
    q = list(map(int, input().split()))
    queries[i] = q
    b = q[1]
    children[b].append(i)

class RollbackDSU:
    def __init__(self, n):
        self.p = list(range(n))
        self.sz = [1] * n
        self.diff = [0] * n
        self.st = []

    def find(self, x):
        d = 0
        while self.p[x] != x:
            d = (d + self.diff[x]) % K
            x = self.p[x]
        return x, d

    def snapshot(self):
        return len(self.st)

    def rollback(self, snap):
        while len(self.st) > snap:
            t = self.st.pop()
            if t[0] == 0:
                continue
            _, b, pb, sb, db = t
            self.p[b] = pb
            self.sz[pb] = sb
            self.diff[b] = db

    def unite(self, x, y, w):
        rx, dx = self.find(x)
        ry, dy = self.find(y)
        if rx == ry:
            return (dy - dx) % K == w
        if self.sz[rx] < self.sz[ry]:
            rx, ry = ry, rx
            dx, dy = dy, dx
            w = (-w) % K
        self.st.append((1, ry, self.p[ry], self.sz[rx], self.diff[ry]))
        self.p[ry] = rx
        self.sz[rx] += self.sz[ry]
        self.diff[ry] = (dx + w - dy) % K
        return True

dsu = RollbackDSU(N + 1)
ans = []

def dfs(v):
    snap = dsu.snapshot()
    q = queries[v]
    if q[0] == 0:
        _, _, L, R, X = q
        ok = dsu.unite(L - 1, R, X)
        ans.append("YES" if ok else "NO")
    else:
        _, _, L, R = q
        a, da = dsu.find(L - 1)
        b, db = dsu.find(R)
        if a != b:
            ans.append("UNKNOWN")
        else:
            ans.append(str((db - da) % K))
    for to in children[v]:
        dfs(to)
    dsu.rollback(snap)

for to in children[0]:
    dfs(to)

print("\n".join(ans))
```

## N. Stock Price Correction

### 中文题面
给定序列 \(H\)，要把它改成严格递增整数序列 \(H'\)，使  
\[
\sum |H_i - H'_i|
\]
最小，求最小值。

### 思路
令 \(B_i = H_i - i\)，问题转化为把 \(B\) 拟合成非递减序列，最小化 L1 距离。  
可用单调分块 / PAV 算法。

### Python 代码
```python
import sys
input = sys.stdin.readline

n = int(input())
h = list(map(int, input().split()))

blocks = []
for i, x in enumerate(h, 1):
    blocks.append([x - i])

    while len(blocks) >= 2:
        if blocks[-2][-1] <= blocks[-1][0]:
            break
        a = blocks.pop()
        b = blocks.pop()
        m = []
        i1 = i2 = 0
        while i1 < len(b) and i2 < len(a):
            if b[i1] < a[i2]:
                m.append(b[i1]); i1 += 1
            else:
                m.append(a[i2]); i2 += 1
        m.extend(b[i1:])
        m.extend(a[i2:])
        blocks.append(m)

def block_cost(arr):
    med = arr[len(arr)//2]
    return sum(abs(x - med) for x in arr)

ans = 0
for arr in blocks:
    ans += block_cost(arr)

print(ans)
```

## O. Joining of Circular Tablets

### 中文题面
给定环形数组，每次只能合并相邻两块，代价为两块重量之和，合并后新块重量仍为总和。  
求合并成一块的最小总代价。

### 思路
环形石子合并，区间 DP；复制数组一遍后做线性区间 DP。

### Python 代码
```python
import sys
input = sys.stdin.readline

n = int(input())
a = list(map(int, input().split()))
b = a + a
pre = [0] * (2 * n + 1)
for i in range(2 * n):
    pre[i + 1] = pre[i] + b[i]

def s(l, r):
    return pre[r + 1] - pre[l]

INF = 10**30
dp = [[0] * (2 * n) for _ in range(2 * n)]
opt = [[0] * (2 * n) for _ in range(2 * n)]

for i in range(2 * n):
    opt[i][i] = i

for length in range(2, n + 1):
    for l in range(0, 2 * n - length + 1):
        r = l + length - 1
        dp[l][r] = INF
        start = opt[l][r - 1]
        end = opt[l + 1][r] if l + 1 <= r else r
        if start < l:
            start = l
        if end > r - 1:
            end = r - 1
        bestk = start
        for k in range(start, end + 1):
            val = dp[l][k] + dp[k + 1][r] + s(l, r)
            if val < dp[l][r]:
                dp[l][r] = val
                bestk = k
        opt[l][r] = bestk

ans = INF
for l in range(n):
    ans = min(ans, dp[l][l + n - 1])

print(ans)
```
