### A. Calculating Part-Time Job Pay / 兼职工资计算
题意：求总工资。  
思路：直接累加 \(a_i \times t_i\)。
```python
import sys
input = sys.stdin.readline

n = int(input())
ans = 0
for _ in range(n):
    a, t = map(int, input().split())
    ans += a * t
print(ans)
```

### C. Permission Management System / 权限管理系统
题意：查询若干资源需求的按位并。  
思路：答案就是这些二进制串按位 OR。
```python
import sys
input = sys.stdin.readline

n, L, q = map(int, input().split())
s = [int(input().strip(), 2) for _ in range(n)]

for _ in range(q):
    tmp = list(map(int, input().split()))
    x = 0
    for i in tmp[1:]:
        x |= s[i - 1]
    print(format(x, f'0{L}b'))
```

### D. Team Division / 队伍分割
题意：把数组分成两段，最小化两段和差。  
思路：枚举分割点，取最小绝对差。
```python
import sys
input = sys.stdin.readline

n = int(input())
a = list(map(int, input().split()))
tot = sum(a)
pre = 0
ans = 10**18
for i in range(n - 1):
    pre += a[i]
    ans = min(ans, abs(tot - 2 * pre))
print(ans)
```

### F. Sales Target for Consecutive Intervals / 连续区间销售目标
题意：统计和至少为 K 的连续区间数。  
思路：双指针。
```python
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
a = list(map(int, input().split()))

r = s = ans = 0
for l in range(n):
    while r < n and s < k:
        s += a[r]
        r += 1
    if s >= k:
        ans += n - r + 1
    s -= a[l]
print(ans)
```

### G. Circle of Friends / 朋友圈
题意：询问某点所在连通块大小。  
思路：并查集维护大小。
```python
import sys
input = sys.stdin.readline

class DSU:
    def __init__(self, n):
        self.p = list(range(n))
        self.sz = [1] * n
    def find(self, x):
        while self.p[x] != x:
            self.p[x] = self.p[self.p[x]]
            x = self.p[x]
        return x
    def union(self, a, b):
        a, b = self.find(a), self.find(b)
        if a == b:
            return
        if self.sz[a] < self.sz[b]:
            a, b = b, a
        self.p[b] = a
        self.sz[a] += self.sz[b]

n, m = map(int, input().split())
d = DSU(n)
for _ in range(m):
    u, v = map(int, input().split())
    d.union(u - 1, v - 1)

q = int(input())
for _ in range(q):
    x = int(input()) - 1
    print(d.sz[d.find(x)])
```

### H. City Tour Survey / 城市巡查
题意：按规则输出访问顺序。  
思路：优先队列维护当前可达点里重要度最大的。
```python
import sys
from heapq import heappush, heappop
input = sys.stdin.readline

n, m = map(int, input().split())
b = list(map(int, input().split()))
g = [[] for _ in range(n)]
for _ in range(m):
    u, v = map(int, input().split())
    g[u - 1].append(v - 1)

vis = [False] * n
ans = []
pq = []

def add(x):
    if vis[x]:
        return
    vis[x] = True
    ans.append(x + 1)
    for y in g[x]:
        if not vis[y]:
            heappush(pq, (-b[y], y))

add(0)
while len(ans) < n:
    while pq and vis[pq[0][1]]:
        heappop(pq)
    if pq:
        _, x = heappop(pq)
        add(x)
    else:
        x = max((i for i in range(n) if not vis[i]), key=lambda i: b[i])
        add(x)

print(*ans)
```

### I. Circle Pass Game / 圈形传球
题意：圆圈传球，模拟最后位置。  
思路：树状数组找第 k 个存活者。
```python
import sys
input = sys.stdin.readline

class BIT:
    def __init__(self, n):
        self.n = n
        self.bit = [0] * (n + 1)
    def add(self, i, v):
        while i <= self.n:
            self.bit[i] += v
            i += i & -i
    def sum(self, i):
        s = 0
        while i > 0:
            s += self.bit[i]
            i -= i & -i
        return s
    def kth(self, k):
        i = 0
        step = 1 << (self.n.bit_length() - 1)
        while step:
            ni = i + step
            if ni <= self.n and self.bit[ni] < k:
                k -= self.bit[ni]
                i = ni
            step >>= 1
        return i + 1

n, m, s = map(int, input().split())
d = [int(input()) for _ in range(m)]

bit = BIT(n)
for i in range(1, n + 1):
    bit.add(i, 1)

cur = s
for x in d:
    alive = bit.sum(n)
    pos = bit.sum(cur)
    k = pos + x
    if k > alive - 1:
        k = (k - 1) % (alive - 1) + 1
    nxt = bit.kth(k + (1 if k >= pos else 0))
    bit.add(cur, -1)
    cur = nxt

print(cur)
```
