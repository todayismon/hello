- 漂亮的牡丹花(flower)
【题目描述】
洛阳市盛开了很多漂亮的牡丹花，有 2000001 朵牡丹花排成一列，编号从 −1000000 到 1000000，其中有 n 朵连续的牡丹花是白色，如果编号为 k 的牡丹花 是白色，请按升序输出所有可能为白色牡丹花的编号。
【输入格式】
输入一行两个整数，分别为 n 和 k，其中 n 表示有 n 朵连续的白色牡丹花，k 表示已知第 k 朵牡丹花是白色的。
【输出格式】
输出一行可能为白色牡丹花的编号，每两个编号之间用空格隔开。

【样例 1 输入】
3 7
【样例 1 输出】
5 6 7 8 9 
【样例 1 解释】有连续的 3 朵牡丹花是白色，已知编号为 7 的牡丹花是白色时，编号为 5、6、7、8、9 的牡丹花都有可能是白色。

【样例 2 输入】
4 0
【样例 2 输出】
-3 -2 -1 0 1 2 3 
【样例 2 解释】有连续的 4 朵牡丹花是白色，已知编号为 0 的牡丹花是白色时，编号为-3、-2、 -1、0、1、2、3 的牡丹花都有可能是白色。

【样例 3 输入】
1 100 
【样例 3 输出】
100
【样例 3 解释】有连续的 1 朵牡丹花是白色，已知编号为 100 的牡丹花是白色时，可能是白色牡丹花的只有编号 100。

【数据范围】
1 ≤ n ≤ 100
0 ≤ k ≤ 100

- 神奇的地图(plat)
【题目描述】

//plat (AT_abc175_e)
#include <bits/stdc++.h>
using namespace std;
long long v[3005][3005];

int main(){
        cin.tie(0) -> sync_with_stdio(0);
        int n, m, k;
        cin >>n >>m >>k;
        for(int i=1;i<=k;i++){
                int x, y;
                cin >>x >>y >>v[x][y];
        }
        long long t1,t2,t3; // take1, take2, take3
        for(int x=1;x<=n;x++){
                t1=t2=t3=0;
                for(int y=1;y<=m;y++){
                        t3= max(t3, t2+v[x][y]);
                        t2= max(t2, t1+v[x][y]);
                        t1= max(t1, v[x-1][y]+v[x][y]);
                        v[x][y]= max(t1,max(t2,t3));
                }
        }
        cout << v[n][m]<<'\n';
        return 0;
}
